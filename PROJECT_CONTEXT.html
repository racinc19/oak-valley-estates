📦 Hand-Off Workflow

A "hand-off" is a complete package of documents delivered to a subcontractor or client.

Standard Package Contents

  • Contract / agreement document (DOCX/PDF)
  • Relevant plans (PDF/DWG)
  • Renderings or visual references (PNG/JPG)
  • Selections or specs (HTML/CSV/PDF)
  • Any required COI or license docs

Steps

  1. Gather all files from Construction/, Documents/, Trades/
  2. Verify completeness — read the contract, check plan count, confirm rendering coverage
  3. Create zip with naming convention: {Trade}_{PackageType}_Rodriguez.zip
  4. Document contents in a summary text block (for email body)
  5. Deliver via: email, OVE subs site upload, or shared drive
  6. Verify delivery — confirm receipt, check sent folder, confirm upload success
CRITICAL: Never report "sent" or "uploaded" without verification. If you cannot confirm delivery, say "I attempted but cannot verify delivery."
