Oak Valley Estates (OVE) — Contour data for Rodriguez / Lot 5

Files in this folder:
- OVE_revit.csv   — X,Y,Z point data for Revit or other tools (elevation in third column). Large point set.
- OVE_final.csv   — Contour polylines with fid, ID, ELEV, vertex_index, distance, angle, X, Y. ELEV in feet (~1651–1664). Use for GIS or Revit contour import.

The previous OVE_final.qmd was an empty QGIS project and was removed. To use in QGIS, add OVE_final.csv or OVE_revit.csv as a layer in a new project.
